## 📋 Zetamed – Backend (Envio de E-mails)

Backend responsável exclusivamente pelo envio de e-mails do site da Zetamed, com foco em segurança, validação de dados e proteção contra bots.

Este serviço recebe os dados do formulário de contato do site e realiza o envio de e-mails de forma controlada, evitando spam e tentativas automatizadas.

## 🔍 Funcionalidades

✅ Endpoint para envio de e-mails do formulário de contato

✅ Proteção contra bots e requisições automatizadas

✅ Tratamento de erros e respostas padronizadas

✅ Integração direta com o frontend do site
